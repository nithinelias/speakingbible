What makes SpeakingBible unique ?

It is NOT

a Bible Audio Player

  • SpeakingBible is not a mp3 player with Bible audio content.
  • It does not follow a recitation model with limitations for bible meditation.
  • Ofcourse, these MP3 players are hardy and low cost.

It is NOT

a Bible App for the literate

  • Most commonly, BibleApps expect a bilingual/literate user to access language specific content.
  • Text and audio function independently of each other in these Apps.

It IS

A Bible App for the illiterate

  • SpeakingBible enables the illiterate user to interact with scripture in detail.
  • It helps the user to become intimate with Scripture by syncing audio with text transcripts.
  • The User Experience design models scripture meditation for the illiterate.
